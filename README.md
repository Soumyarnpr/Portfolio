# Soumya Ranjan Parida Portfolio

A Next.js, React, Tailwind CSS, and GSAP portfolio app converted from the original static HTML/CSS/JavaScript site.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Structure

- `src/app`: Next.js App Router entry files and global styles.
- `src/components`: Reusable portfolio sections and animation components.
- `src/data/portfolio.js`: Projects, skills, profile, navigation, and contact content.
- `public/images`: Static images, favicon, and resume assets.
